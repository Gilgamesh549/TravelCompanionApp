# Travel Companion App (SIT305 Task 2.1)

## Description
This is an Android application developed for SIT305/SIT708 Task 2.1. It helps international travellers convert essential values.

## Features
* **Currency Conversion:** Converts USD to AUD, EUR, JPY, and GBP.
* **Fuel & Distance:** Converts mpg to km/L, Gallons to Liters, and Nautical Miles to Kilometers.
* **Temperature:** Converts between Celsius, Fahrenheit, and Kelvin.
* **Validation:** Includes robust error handling to prevent empty inputs, handle identity conversions, and reject invalid negative numbers.

## Tools Used
* Android Studio
* Kotlin
